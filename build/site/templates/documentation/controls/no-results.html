No Results Control

This control represents a text or HTML that is shown after filtering when no results are found.

HTML Structure

No results control is defined by required data-jplist-control="no-results" data attribute. The data-group attribute defines group of elements that should be filtered. Multiple not found controls are allowed. 

<!-- different jplist filter controls -->
...

<!-- content to filter -->
<div data-jplist-group="group1">

    <!-- item -->
    <div data-jplist-item>
        <div class="title">Title value 1</div>
        ....
    </div>

    <!-- item -->
    <div data-jplist-item>
        <div class="title">Title value 1</div>
        ....
    </div>

    <!-- item -->
    <div data-jplist-item>
        <div class="title">Title value 1</div>
        ....
    </div>
                
    <!-- no results control -->               
    <div data-jplist-control="no-results" data-group="group1" data-name="no-results">No Results Found</div>

</div>

<!-- jplist library -->
<script src="jplist.min.js"></script>

    jplist.init();
</script>
