Reset Control

This control resets all other controls to their initial state.

HTML Structure

Reset control is defined by required data-jplist-control="reset" data attribute. The data-group attribute defines group of elements that should be filtered. 

<!-- different jplist controls -->
...

<!-- reset control -->
<button
        data-jplist-control="reset"
        data-group="group1"
        type="button">
    Reset
</button>
                
<!-- content -->
<div data-jplist-group="group1">

    <!-- item -->
    <div data-jplist-item>
        <div class="title">Title value 1</div>
        ....
    </div>

    <!-- item -->
    <div data-jplist-item>
        <div class="title">Title value 1</div>

    </div>

</div>
